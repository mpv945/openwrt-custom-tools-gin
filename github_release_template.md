## ⚡ 性能优化

-   🚀 **启动优化**：减少初始化时间\
-   🧠 **内存优化**：降低内存占用并修复潜在泄漏\
-   📜 **日志增强**：支持更多调试信息\
-   ⚙️ **配置优化**：配置方式更加灵活

------------------------------------------------------------------------

## 🐛 问题修复

-   修复部分环境 **程序无法启动** 的问题\
-   修复 **配置文件异常导致崩溃** 的问题\
-   修复 **跨平台兼容性问题**\
-   修复 **部分网络请求超时问题**

------------------------------------------------------------------------

## ⚠️ 不兼容变更

> 以下变更可能影响旧版本，请升级时注意, 本地图片：![UI](docs/images/ui.png)

### 配置项变更

`old_config` 已替换为 `new_config`

### API 调整

移除接口：

    v1/old-endpoint

新增接口：

    v2/new-endpoint

### 数据库结构调整

升级时请执行数据库迁移脚本

### ARM64 muslib 兼容问题

upstream 已修复\
https://github.com/oracle/graal/issues/13087

------------------------------------------------------------------------

## 🔄 升级指南

升级步骤：

1.  更新配置文件 `config.yaml`
2.  重新构建项目

``` bash
mvn clean package
```

3.  更新 API 调用路径

### 配置变更示例

旧配置：

``` yaml
old_config: true
```

新配置：

``` yaml
new_config: true
```
