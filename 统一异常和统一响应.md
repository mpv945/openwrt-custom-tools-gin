在使用 Gin 开发企业级 API 时，**统一异常处理 + 统一返回结构**是非常重要的设计。
生产项目通常实现三件事：

1️⃣ **统一返回结构（Response）**
2️⃣ **统一错误处理（Error Handler Middleware）**
3️⃣ **统一 Panic Recovery**

我给你一套 **生产级完整实现**，可以直接用于项目。

# 一、统一返回结构

先定义 API 返回格式，例如：

```json
{
  "code": 0,
  "msg": "success",
  "data": {}
}
```

创建：

```
internal/model/response.go
```

```go
package model

type Response struct {
	Code int         `json:"code"`
	Msg  string      `json:"msg"`
	Data interface{} `json:"data,omitempty"`
}
```

# 二、统一返回工具

创建：

```
internal/response/response.go
```

```go
package response

import (
	"net/http"

	"github.com/gin-gonic/gin"
	"yourproject/internal/model"
)

func Success(c *gin.Context, data interface{}) {

	c.JSON(http.StatusOK, model.Response{
		Code: 0,
		Msg:  "success",
		Data: data,
	})
}

func Error(c *gin.Context, code int, msg string) {

	c.JSON(http.StatusOK, model.Response{
		Code: code,
		Msg:  msg,
	})
}
```

创建：

```

```

```go
package errors

type BizError struct {
	Code int
	Msg  string
}

func (e *BizError) Error() string {
	return e.Msg
}

func New(code int, msg string) *BizError {
	return &BizError{
		Code: code,
		Msg:  msg,
	}
}
```

# 四、统一异常处理中间件

创建：

```
internal/middleware/error_handler.go
```

```go
package middleware

import (
	"net/http"

	"github.com/gin-gonic/gin"
	"yourproject/internal/errors"
	"yourproject/internal/model"
)

func ErrorHandler() gin.HandlerFunc {

	return func(c *gin.Context) {

		c.Next()

		if len(c.Errors) > 0 {

			err := c.Errors.Last().Err

			if bizErr, ok := err.(*errors.BizError); ok {

				c.JSON(http.StatusOK, model.Response{
					Code: bizErr.Code,
					Msg:  bizErr.Msg,
				})

				return
			}

			c.JSON(http.StatusInternalServerError, model.Response{
				Code: 500,
				Msg:  err.Error(),
			})
		}
	}
}
```

# 五、统一 Panic Recovery

虽然 Gin 自带 `Recovery()`，但生产通常自定义（结合日志）。

```go
package middleware

import (
	"net/http"

	"github.com/gin-gonic/gin"
)

func Recovery() gin.HandlerFunc {

	return func(c *gin.Context) {

		defer func() {

			if err := recover(); err != nil {

				c.JSON(http.StatusInternalServerError, gin.H{
					"code": 500,
					"msg":  "internal server error",
				})

				c.Abort()
			}

		}()

		c.Next()
	}
}
```

# 六、Router 注册中间件

```go
package router

import (
	"github.com/gin-gonic/gin"
	"yourproject/internal/middleware"
)

func InitRouter() *gin.Engine {

	gin.SetMode(gin.ReleaseMode)

	r := gin.New()

	r.Use(middleware.Recovery())
	r.Use(middleware.ErrorHandler())

	return r
}
```

# 七、API 示例

import (
	"github.com/gin-gonic/gin"
	"yourproject/internal/errors"
	"yourproject/internal/response"

func GetUser(c *gin.Context) {

	id := c.Query("id")

	if id == "" {
		c.Error(errors.New(1002, "id required"))
		return
	}

	response.Success(c, gin.H{
		"id": id,
	})
}
